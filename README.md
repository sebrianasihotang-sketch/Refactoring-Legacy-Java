# Refactoring Legacy Java Swing

## Deskripsi
Project ini dibuat sebagai simulasi refactoring kode legacy Java Swing.

## Struktur
- entity
- repository
- service
- controller
- database
- utils
- view

## Database
Import file `database.sql` ke MySQL.

## Cara Menjalankan
1. Jalankan MySQL.
2. Import database.sql.
3. Jalankan MainFrame.java.
